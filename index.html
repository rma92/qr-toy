QR Code File Copy

  • Generator
  • Reader

The QR Code File Copy generates a stream of QR codes containing segments of a file or other data. The reader has an HTML5 scanner that uses a camera to continuously scan and store data. Once all the pieces of a file are scanned, the segments are combined and the file can be saved.

Header format

The header in each QR code contains a series of fields separated by colons (:), followed by a double colon (::). The fields are as follows:

  • Q - just an identifying header.
  • Current Chunk ID (integer) - the current chunk. They should be 0 indexed.
  • Total Chunks (integer) - the total number of chunks for this file.
  • CRC32 (integer) - the CRC32 for the entire file.
  • Filename (string, optional) - a file name to save the file as once it's done transferring.
  • encode (string, optional) - B64, or B10 for base64 encoding or base10 encoding, LB1 for base10 encoding of LZMA'd data. (the generator can encode Base64 of an LZMA but this is fairly useless)
